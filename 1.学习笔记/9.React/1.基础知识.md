## 脚手架

```shell
npm install -g create-react-app

D:  //进入D盘
mkdir ReduxDemo   //创建ReduxDemo文件夹
cd ReduxDemo      //进入文件夹
create-react-app demo01  //用脚手架创建React项目
cd demo01   //等项目创建完成后，进入项目目录
npm  start  //预览项目
```
