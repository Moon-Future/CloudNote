#### 圣杯布局

```html
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        html, body {
            height: 100%;
            padding: 0;
            margin: 0;
            min-width: 600px;
        }
        .clearfix:after {
            content: "";
            display: block;
            clear: both;
            visibility: hidden;
            height: 0;
        }
        .left {
            height: 300px;
            background: lightblue;
        }
        .center {
            height: 300px;
            background: red;
        }
        .right {
            height: 300px;
            background: yellow;
        }

        .container {
            padding: 0 200px;
        }

        .left, .right {
            width: 200px;
        }

        .center {
            width: 100%;
        }

        .left, .center, .right {
            float: left;
        }

        .left {
            margin-left: -100%;
            position: relative;
            left: -200px;
        }

        .right {
            margin-right: -200px;
        }
    </style>
</head>
<body>
    <div class="container clearfix">
        <div class="center">中</div>
        <div class="left">左</div>
        <div class="right">右</div>
    </div>
</body>
</html>
```
