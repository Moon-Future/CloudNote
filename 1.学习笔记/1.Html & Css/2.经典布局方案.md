## 三栏布局

### 左右固定，中间自适应

#### 圣杯布局

浮动 + 负 margin

```html
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body {
            min-width: 550px;
        }

        .clearfix:after {
            content: "";
            display: block;
            clear: both;
            visibility: hidden;
            height: 0;
        }

        .container {
            padding-left: 200px;
            padding-right: 150px;
        }

        .container .column {
            float: left;
            height: 200px;
        }

        #center {
            width: 100%;
            background: #eaffd0;
        }

        #left {
            width: 200px;
            margin-left: -100%;
            position: relative;
            left: -200px;
            background: #fce38a;
        }

        #right {
            width: 150px;
            margin-right: -150px;
            background: #95e1d3;
        }
    </style>
</head>
<body>
    <div class="container clearfix">
        <div id="center" class="column">中</div>
        <div id="left" class="column">左</div>
        <div id="right" class="column">右</div>
    </div>
</body>
</html>
```

#### 绝对定位

```html
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        body {
            min-width: 550px;
        }

        .container {
            position: relative;
        }

        .column {
            height: 200px;
        }

        #center {
            margin-left: 200px;
            margin-right: 150px;
            background: #eaffd0;
        }

        #left {
            width: 200px;
            position: absolute;
            top: 0;
            left: 0;
            background: #fce38a;
        }

        #right {
            width: 150px;
            position: absolute;
            top: 0;
            right: 0;
            background: #95e1d3;
        }
        
    </style>
</head>
<body>
    <div class="container">
        <div id="center" class="column">中</div>
        <div id="left" class="column">左</div>
        <div id="right" class="column">右</div>
    </div>
</body>
</html>
```

## 两栏布局

### 左宽度固定，右自适应

#### float + margin

```html
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    .container {
      height: 200px;
    }
    .container div {
      height: 100%;
    }
    .left {
      float: left;
      width: 200px;
      background: blue;
    }
    .right {
      margin-left: 200px;
      background: red;
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="left">左</div>
    <div class="right">右</div>
  </div>
</body>
</html>
```
